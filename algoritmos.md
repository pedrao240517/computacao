# Algoritmos
##Exercicios de algoritmos feito do curso do gustavo guanabara

Olá, o objetivo desse repositorio é salvar e documentar todos os exercícios feito durante o curso de algoritmos-lógica de programação disponibilizado pelo curso em vídeo do gustavo guanabara.

Link do curso:https://www.youtube.com/playlist?list=PLHz_AreHm4dmSj0MHol_aoNYCSGFqvfXV

O curso de Algoritmos do Curso em Vídeo, ministrado pelo mestre Gustavo Guanabara, é considerado a "porta de entrada" definitiva para quem quer aprender a programar no Brasil. Ele foca totalmente no desenvolvimento do raciocínio lógico antes de você se preocupar com a sintaxe complexa de linguagens como Java ou Python.

O curso utiliza o Portugol (uma pseudolinguagem em português) dentro da ferramenta Visualg. Aqui está o resumo do que você aprende:

* Fundamentos e Lógica de Programação
  
Antes de digitar qualquer código, o curso ensina como o computador "pensa".
O que é um algoritmo: A sequência de passos para resolver um problema.
Hardware e Software: Noções básicas de como a memória e o processador lidam com dados.
Variáveis e Tipos de Dados: Como guardar informações usando tipos como Inteiro, Real, Caractere e Logico.
Entrada e Saída: Os comandos básicos para o programa interagir com o usuário (Leia e Escreva).

 *Operadores e Expressões
 
Aqui você aprende a "matemática" e a "gramática" da programação:
Aritméticos: Soma, subtração, multiplicação, divisão, módulo (resto) e exponenciação.
Relacionais: Comparação entre valores (maior, menor, igual, diferente).
Lógicos: O uso de conectivos E, OU e NAO para criar condições complexas.

* Estruturas Condicionais (Tomada de Decisão)
  
Onde o programa começa a fazer escolhas:
Estrutura Simples e Composta: O famoso SE (condição) ENTAO ... SENAO.
Condicionais Aninhadas: "Se dentro de Se" para decisões mais refinadas.
Escolha Caso: Uma estrutura para lidar com múltiplas opções de forma organizada (o switch de outras linguagens).

* Estruturas de Repetição (Loops)
  
Como fazer o computador repetir tarefas sem escrever o código várias vezes:
Enquanto (While): Repete enquanto uma condição for verdadeira (teste lógico no início).
Repita (Do While): Repete até que uma condição seja atingida (teste lógico no fim).
Para (For): Repete um número determinado de vezes usando uma variável de controle/contador.

* Procedimentos e Funções
  
Introdução à Modularização, para deixar o código limpo e organizado:
Procedimentos: Blocos de código que executam uma tarefa (sem retorno).
Passagem de Parâmetros: Como enviar dados para dentro desses blocos (por valor e por referência).
Funções: Semelhantes aos procedimentos, mas que devolvem (retornam) um resultado para quem as chamou.

* Estruturas de Dados (Coleções)
  
Como armazenar muitos dados em uma única variável:
Vetores (Arrays Unidimensionais): Listas de variáveis do mesmo tipo.
Matrizes (Arrays Multidimensionais): Tabelas com linhas e colunas para dados mais complexos.
